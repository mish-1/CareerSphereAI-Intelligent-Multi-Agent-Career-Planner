# Frontend

Next.js dashboard for CareerSphere AI with landing, analysis, upload, opportunities, safety, resume optimization, mentorship, and settings views.

## Run

```bash
npm install
npm run dev
```

## Key Pages

- `/` landing page
- `/dashboard` overview widgets
- `/resume-upload` resume parsing
- `/career-analysis` orchestrated workflow output
- `/opportunities` job and internship recommendations
- `/safety-reports` workplace safety insights
- `/resume-optimizer` ATS optimization workflow
- `/mentor-chat` mentorship guidance
- `/settings` environment and auth controls
